# Heyi
Programming to solve the unity algorithm
通过C++编程求解合一算法。文字L1和L2如果经过执行某个置换s，满足L1s=L2s，则称L1与L2可合一，s称为其合一元。本程序可判断任意两个文字能否合一，若能合一，则给出其合一元。要求有运行界面！

程序的实现：
Transform 中的t_f1，t_f2 分别表示差异集中的两个字符串。
Vector 变量lan 存放合一的集合。
功能函数：
Issyn() 判断两个公式是否相同。
different() 求差异集。
change() 用代换对公式进行合一代换。
legal() 对t_f1,t_f2 判断然后对换并判断是否为合一字符串
same() 判断两个公式是否相同。
var() 字符串每个()内的子串是变量还是常量。
show() 最终演示算法函数。

程序的具体执行过程：
程序执行时，先输入公式集合中的公式，然后调用Issyn 函数判断是否能合一，如果不能则退出，如果能，判断公式是否相同，相同则合一是ε，不相同则进一步判断。调用different 函数找出差异集放在vector变量 lan 中再调用legal 函数和var 函数判断合一时到底哪个是变量，哪个是常量。t_f1 存放常量，t_f2 存放变量，不符合则进行交换，用t_f1 代换掉公式中的t_f2进行下一轮的比较，寻找差异集然后判断两个公式是否相同再代换。每次差异集都添加到lan 中最后输出lan 中的元素（即合一式）。
